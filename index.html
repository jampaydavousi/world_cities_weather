Summary

This project utilizes Python, Pandas, API’s and JSON traversals to illustrate answers to a fundamental question: “What’s the weather like as we get closer to the equator?”

Randomly selecting 4,000 city Latitude and Longitudes by importing CitiPy library to Pandas, and using the OpenWeatherMapAPI to get the weather data for the cities we were left with list of about 1,200 cities with weather data, after eliminating duplicates and cities with no weather data.

The cities' latitude and longitude along with weather data was placed and organized in a pandas dataframe and exported as csv file. The plot above shows the 1200 cities was created using matpoltlib - almost tracing the land mass of the planet. The other plots also created using matplotlib, illustrate how cities' weather attributes maximumtemperature, humidity, cloudiness, and wind speed change at different latitudes.

  • Temperature (F) vs. Latitude
  • Humidity (%) vs. Latitude
  • Cloudiness (%) vs. Latitude
  • Wind Speed (mph) vs. Latitude
